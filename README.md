# React Tutorial

There is a simple node server implementation that serves static files from `public/`.


### Dependencies

Make sure you have node and npm installed. 

```sh
brew install node
brew update

node -v
npm -v
```

### To Start Server

```sh
npm install
npm start
```

Visit <http://localhost:3000/>.


### To Compile JSX to Javascript While Developing

```sh
gulp
```
